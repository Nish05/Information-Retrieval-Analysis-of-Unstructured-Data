Steps to run the code:
1) Unzip the folder
2) Compile and run the Parser.java file
3) Enter the path for folder and stoplist file in the below format:
   <path of folder> <path of stop list file>
   View the console for inverted index
4) Select an option from the Menu:
   1 - to provide the input of your choice
   2 - to run the test cases
5) View the console for the search results
